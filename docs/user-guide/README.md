## üí° Tips for Success

### Start Small
- Begin with low density (0.1) and small lattice
- Increase complexity gradually
- Test configurations on short runs first

### Use the Tools
- Web interface for exploration
- Command line for automation
- Benchmarks for comparison

### Read the Docs
- Check FAQ before asking questions
- Use Glossary for unfamiliar terms
- Follow tutorials step-by-step

### Get Help
- Review [SUPPORT](../../SUPPORT.md) for help options
